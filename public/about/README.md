# About Page Assets

This directory contains assets for the about page.

## Required Files

### Profile Image
- `profile.jpg` - Main profile photo for about section (recommended: 800x800px)

## Optional Files

### Skill Icons (if using custom icons)
- `skill-react.svg`
- `skill-nodejs.svg`
- etc.

## Usage

These files are referenced in `data/about.ts`:

```typescript
image: '/about/profile.jpg',
```

## Image Guidelines

- Format: JPG, PNG, or WebP for photos; SVG for icons
- Profile image should be high quality
- Recommended size: 800x800px or larger
- Optimize for web performance
