# Certifications Images Directory

This directory contains certificate images for the certifications displayed on the Certifications page.

## Image Guidelines

### Format
- **Recommended Format**: JPG or PNG
- **Recommended Size**: 1200x900px minimum
- **Aspect Ratio**: 4:3 or 16:9 for best display

### Naming Convention
Use descriptive names that match the certification:
- `aws-saa.jpg` - AWS Solutions Architect Associate certificate
- `aws-saa-badge.jpg` - AWS digital badge
- `gcp-pca.jpg` - Google Cloud Professional Cloud Architect
- `azure-expert.jpg` - Azure Solutions Architect Expert
- `cka.jpg` - Certified Kubernetes Administrator
- `terraform.jpg` - HashiCorp Terraform Associate
- `terraform-badge.jpg` - HashiCorp digital badge

## Features

### Image Slider
- Certifications can have multiple images
- Users can navigate through images using:
  - Left/Right arrow buttons
  - Dot indicators at the bottom
  - Direct click on any dot

### Image Display
- **Card View**: Shows the first image as a preview
- **Modal View**: Full image slider with navigation controls
- **Image Count Badge**: Shows "X Images" when multiple images exist

## Tips for Best Results

1. **High Quality**: Use high-resolution images for professional appearance
2. **Consistent Style**: Try to maintain similar styling across all certificates
3. **Clear Text**: Ensure certificate text is readable in the preview
4. **Multiple Views**: For certifications with badges, include both the full certificate and the badge
