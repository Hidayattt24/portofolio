# Home Page Assets

This directory contains assets for the home page.

## Required Files

### Avatar
- `avatar.jpg` - Profile picture/avatar (recommended: 400x400px, square)

### Resume
- `resume.pdf` - Your resume/CV file

## Usage

These files are referenced in `data/home.ts`:

```typescript
avatar: '/home/avatar.jpg',
resumeUrl: '/home/resume.pdf',
```

## Image Guidelines

- Format: JPG, PNG, or WebP
- Avatar should be square (1:1 aspect ratio)
- Optimize images for web (compress without losing quality)
- Use descriptive file names
